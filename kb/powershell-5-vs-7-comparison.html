When to Use Which

Stay on 5.1

  • Legacy vendor modules without Core support
  • Workflows still in production use
  • Locked-down Windows Server without pwsh deployed
  • DSC v1 pull server infrastructure
  • Old Exchange on-prem snap-ins

Move to 7

  • Cross-platform scripts (CI/CD, Linux, containers)
  • Performance-sensitive automation
  • New development — 7 is the forward path
  • Parallel processing, SSH remoting, modern syntax
  • Azure/M365 automation via modern Az modules
